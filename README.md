# Roco Shiny Tracker - 洛克王国世界异色追踪器

一款用于记录《洛克王国：世界》异色精灵刷取进度的 Web PWA 应用。

在线访问：https://zzixxxx.github.io/roco-shiny-tracker/

## 核心特性

### 双池计数系统
游戏中异色精灵的保底机制基于三个独立卡池，本工具重点追踪：
- **家族池** - 持续捕捉同一进化链精灵触发，出该家族污染精灵
- **属性池** - 持续捕捉同属性不同家族精灵触发，出赛季奇遇污染精灵
- 每个池子各自独立计算 80 次噩梦枷锁保底
- 出异色后自动清空对应卡池，保底重新计算

### 高效记录模式
- **褪色结果** - 点击异色/污染/普通直接记录一条（快速连击）
- **批量补录** - 刷完一轮后填入各结果数量，一键记录
- **连续捕捉** - 支持直接输入数字编辑，记录每次触发前的捕捉次数

### 首页仪表盘
- S1赛季异色收集进度总览（19种异色精灵）
- 当前刷取目标（异色立绘 + 属性图标）
- 家族池噩梦枷锁进度条
- 最近获得的异色记录

### 异色图鉴
- 19种异色精灵网格展示（Wiki 异色形态立绘）
- 按分类筛选：全部 / 已收集 / 未收集 / 赛季限定 / 常驻
- 点击查看详情：**普通形态 / 异色形态**双图对比
- 属性图标 + 多色属性名称（双属性各显各色）
- 进化链、出没区域、庇护所果实组合刷取提示

### 刷取日志
- 噩梦枷锁结果时间线（家族池/属性池标识 + 捕捉触发次数）
- **三层筛选**（AND 逻辑）：褪色结果 / 卡池类型 / 精灵
- 点击日志可编辑（修改结果、捕捉数、备注）
- 删除最新记录自动回滚对应卡池计数
- 孵蛋记录管理

### 道具消耗
- 图片网格选择道具（Wiki 球类图片）
- 直接输入数量添加，每条记录可编辑/删除
- 消耗汇总面板

### 数据统计
- 异色出现率、褪色结果分布图
- 各精灵刷取排行
- 道具消耗汇总
- 孵蛋统计
- 数据导入/导出（JSON 备份）

### 云端同步（GitHub Gist）
- 绑定 UID + GitHub Token，数据自动同步到私有 Gist
- 每次记录后 3 秒自动上传（静默同步）
- 换设备：输入同一 Token + UID 即可恢复
- 全局右上角 UID 绑定/解绑

## 技术栈

- **Vue 3** + **Vite** + **Vue Router** + **Pinia**
- **LocalStorage** 本地持久化 + **GitHub Gist** 云端同步
- **PWA** 支持（可添加到手机桌面）
- 全套统一风格 SVG 图标（渐变线条风格）
- 精灵立绘 / 道具图片 / 属性图标：[Bilibili Wiki](https://wiki.biligame.com/rocom/)
- GitHub Pages 自动部署

## 快速开始

```bash
npm install
npm run dev      # 开发
npm run build    # 构建
```

## 云端同步使用

1. 在任意页面右上角点击"绑定UID"，输入你的游戏昵称
2. 进入"数据统计"页面底部"云端同步"
3. 点击"去创建"生成 GitHub Token（勾选 gist 权限）
4. 粘贴 Token，之后每次操作自动同步
5. 换设备：输入同一 Token + UID → "从云端恢复"
