# Roco Shiny Tracker - 洛克王国世界异色追踪器

一款用于记录《洛克王国：世界》异色精灵刷取进度的 Web PWA 应用。

在线访问：https://zzixxxx.github.io/roco-shiny-tracker/

## 核心特性

### 三池并行计数系统
游戏中异色精灵的保底机制基于三类累积池，每次捕捉**3 池同时累积**：
- **家族池** - 按当前刷取目标精灵的进化链累积，到 80 次保底
- **系别池** - 按野怪主属性累积（双系别只算第 1 系别，官方 2026-04-23 公告钦定）
- **大世界池** - 每次捕捉无差别累积（混沌池）
- 每个池子各自独立到 80 次保底，**最先满的先触发**
- 触发后只清空被触发的池，另外两池继续累积（对齐服务端机制）
- 数据来源：`BONUS_EVENT_ACCU_POOL_CONF` 38 个 `accumulate_type`（19 EVO + 18 SKILLDAM + 1 CHAOS），完整对比见 `roco-shiny-tracker-机制对比与建议.md`

### 高效记录模式
- **褪色结果** - 点击异色/污染/普通直接记录一条（快速连击）
- **批量补录** - 刷完一轮后填入各结果数量，一键记录
- **连续捕捉** - 支持直接输入数字编辑，记录每次触发前的捕捉次数

### 首页仪表盘
- 异色收集进度总览（跨赛季）
- 当前刷取目标（异色立绘 + 属性图标）
- 三池噩梦枷锁进度条（家族池 / 系别池 / 大世界池）
- 最近获得的异色记录

### 异色图鉴
- **赛季 tab 切换**（最新赛季在前，默认显示；已支持 S1 / S2）
- 网格展示当前赛季异色精灵 + 按分类筛选（全部 / 已收集 / 未收集 / 赛季限定 / 常驻）
- 进度条按赛季独立统计
- 立绘三段回退：SHINY_PETS 异色 Wiki 图 → BREEDING_PETS 本地普通立绘 → 首字母占位
- 点击查看详情：**普通形态 / 异色形态**双图对比
- 属性图标 + 多色属性名称（双属性各显各色）
- 进化链、出没区域、庇护所果实组合刷取提示

### 刷取日志
- 噩梦枷锁结果时间线（家族池/系别池/大世界池标识 + 捕捉触发次数）
- **三层筛选**（AND 逻辑）：褪色结果 / 卡池类型 / 精灵
- 点击日志可编辑（修改结果、捕捉数、备注）
- 删除最新记录自动回滚对应卡池计数
- 孵蛋记录管理

### 蛋工具集（统一在 `/#/egg`，顶部 tab 切换）

#### 蛋组查询（默认 tab）
- 选一只目标精灵，列出可一起生蛋的所有精灵（egg_group 数组有交集）
- 可选填配偶名，蛋组不匹配显示"No！他们不能在一起"
- "仅看异色" chip 筛选 + 异色立绘切换
- 点击查询结果卡片可切换为新目标
- 数据：`BREEDING_PETS`（177 只 stage 1 可家园生蛋精灵），按 PETBASE_CONF.egg_group 反查

#### 孵蛋时间（`?tab=hatch`）
- 174 只可生蛋精灵
- **搜索**：中文名 / 拼音 / 后缀变体名（如"急急鸭""燃了鸭"）
- **属性筛选**：复用图鉴页的属性图标
- **档位筛选**：5 分钟 / 8h / 12h / 16h / 20h
- **异色筛选**：全部 / 仅看异色（仅看异色时切换异色立绘）
- **加速叠加**：风场 ×6 / 周末 ×2 / 满叠 ×12，乘法叠加

#### 生蛋规划（`?tab=plan`）
**异色传递（自动初始化 + 可视化）：**
- **总图「全部异色」**（默认）：跨赛季所有异色按蛋组分布；顶部 chip 按 S1/S2 筛选 — 不过滤精灵，非当前赛季 dim（灰度+透明）保留布局位置
- 多张传递图 tab 切换（点 "+ 添加传递" 默认起名「自定义N」）
- SVG 自动布局算法：
  - 中心组 = 跨组精灵最多的；成员按「目的地组扩展宽度」分簇（小目的地占少行，大目的地腾空间）
  - 非桥成员同行被占时 ring 搜索就近空格（偏好下方+右侧，避免跳过整行）
  - Isolated 组填主区空隙（8 邻居最少 + 距中心最近）
- **全局添加精灵**：顶部搜索框（中文/拼音），按 BREEDING_PETS.eggGroup 自动加到对应所有蛋组；重名精灵带括号区分（如「鸭吉吉（急急鸭）」）
- **拖动精灵**到任意网格位置自定义布局（interact.js + 网格吸附），<kbd>Ctrl+Z</kbd> 或工具栏 ↶ 撤销
- 编辑模式可改组名 / 加减精灵
- 导出/导入：JSON 模板 + SVG 矢量图（精灵图 inline base64） + PNG 位图（2× 清晰度）
- 大图弹窗：默认 ⤢ 自适应屏幕；支持 +/− 缩放（30%-300%） + 1:1 重置 + ↶ 撤销

**培育工坊：**
- 记录每日家园生蛋（日期、父母、结果、备注）
- 自动分析能否配对（实时查双方蛋组交集）
- 数据 localStorage 持久化

### 道具消耗
- 图片网格选择道具（Wiki 球类图片）
- 直接输入数量添加，每条记录可编辑/删除
- 消耗汇总面板

### 数据统计
- 异色出现率、褪色结果分布图
- 各精灵刷取排行
- 道具消耗汇总
- 孵蛋统计
- 数据导入/导出（JSON 备份）

### 云端同步（GitHub Gist）
- 绑定 UID + GitHub Token，数据自动同步到私有 Gist
- 每次记录后 3 秒自动上传（静默同步）
- 换设备：输入同一 Token + UID 即可恢复
- 全局右上角 UID 绑定/解绑

## 技术栈

- **Vue 3** + **Vite** + **Vue Router** + **Pinia**
- **interact.js** —— SVG 精灵拖拽 + 网格吸附（异色传递图）
- **LocalStorage** 本地持久化 + **GitHub Gist** 云端同步
- **PWA** 支持（可添加到手机桌面）
- SVG 图标 + aoe-top 解包资源（webp icons/friends 全量镜像）
- 精灵立绘 / 道具图片 / 属性图标：[Bilibili Wiki](https://wiki.biligame.com/rocom/)
- GitHub Pages 自动部署

## 快速开始

```bash
# 首次克隆需要拉取游戏配置数据 submodule
git submodule update --init --recursive

npm install
npm run dev      # 开发（predev 钩子会自动 sync 数据到 public/data/）
npm run build    # 构建（prebuild 钩子同样）
```

## 云端同步使用

1. 在任意页面右上角点击"绑定UID"，输入你的游戏昵称
2. 进入"数据统计"页面底部"云端同步"
3. 点击"去创建"生成 GitHub Token（勾选 gist 权限）
4. 粘贴 Token，之后每次操作自动同步
5. 换设备：输入同一 Token + UID → "从云端恢复"
